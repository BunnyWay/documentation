<Update label="March 18, 2026" tags={["New"]}>
  ## Edge rule pattern matching

  You can now use Lua-based pattern matching in Edge Rule conditions to match structured request values such as URLs, headers, cookies, and query strings. [Learn more](/cdn/edge-rules/pattern-matching)
</Update>

<Update label="March 9, 2026" tags={["New"]}>
  ## JA4 fingerprinting

  Identify clients using TLS fingerprinting with the new JA4 fingerprinting feature. The JA4 fingerprint is available via the `CDN-JA4` request header for use in bot detection and security rules. [Learn more](/cdn/security/ja4-fingerprinting)
</Update>

<Update label="March 9, 2026" tags={["Improvement"]}>
  ## Updated logging documentation

  The CDN logging overview and permanent log storage documentation have been updated with improved clarity and additional details. [Learn more](/cdn/logging)
</Update>
