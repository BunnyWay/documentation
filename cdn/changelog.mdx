<Update label="2026-03-18" tags={["New"]}>
  ## Edge rule pattern matching

  You can now use Lua-based pattern matching in Edge Rule conditions to match structured request values such as URLs, headers, cookies, and query strings for advanced traffic routing and security logic.

  [Learn more](/cdn/edge-rules/pattern-matching)
</Update>

<Update label="2026-03-09" tags={["New"]}>
  ## JA4 TLS fingerprinting

  JA4 fingerprinting is now available for CDN Pull Zones. Identify clients by their TLS handshake characteristics using the `CDN-JA4` request header to detect bots, automated traffic, and strengthen your security decisions.

  [Learn more](/cdn/security/ja4-fingerprinting)
</Update>

<Update label="2026-03-09" tags={["Improvement"]}>
  ## Updated logging and permanent log storage

  The CDN logging overview and permanent log storage documentation have been refreshed with clearer configuration details, updated rotation thresholds, and improved guidance for setting up log archival to Edge Storage.

  [Learn more](/cdn/logging)
</Update>
