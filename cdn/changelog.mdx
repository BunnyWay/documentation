<Update label="March 2026" tags={["New"]}>
  ## Edge rule pattern matching

  You can now use advanced pattern matching syntax in edge rules for more precise control over request routing and transformations.

  [Learn more](/cdn/edge-rules/pattern-matching)

<Update label="March 2026" tags={["New"]}>
  ## JA4 fingerprinting

  JA4 fingerprinting is now available as a CDN security feature, giving you deeper visibility into client connections and helping identify suspicious traffic patterns.

  [Learn more](/cdn/security/ja4-fingerprinting)
</Update>

  [Learn more](/cdn/troubleshooting/502-504-errors)
</Update>
