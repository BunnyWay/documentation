<Update label="2026-02-11" tags={["New"]}>
  ## Node.js file system API

  Edge Scripts now support the Node.js-compatible file system API (`node:fs` and `node:fs/promises`), allowing you to read, write, and manipulate files within a sandboxed virtual file system at the edge.

  [Learn more](/scripting/node/fs/node-fs)
</Update>

<Update label="2026-01-19" tags={["New"]}>
  ## WebSocket support

  Edge Scripts now support WebSockets for low-latency, bidirectional real-time communication between your applications and users across the bunny.net CDN.

  [Learn more](/scripting/websockets)
</Update>
