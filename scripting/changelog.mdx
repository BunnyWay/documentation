<Update label="2026-06-17" tags={["New"]}>
  ## Weighted traffic splitting & geo-routing examples

Two new standalone Edge Scripting examples: [weighted traffic splitting](/scripting/standalone/examples/weighted-routing) for sending a percentage of visitors to a second origin with sticky routing, and [route by country](/scripting/standalone/examples/geo-routing) for directing visitors from specific countries to a different origin.

</Update>

<Update label="2026-03-26" tags={["New"]}>
  ## Node.js file system API

A Node.js-compatible file system API is now available in Edge Scripts, allowing you to read and write files directly at the edge. [Learn more](/scripting/node-fs)

</Update>
