<Update label="February 2026" tags={["New"]}>
  ## Node.js File System API

  Edge Scripts now support the `node:fs` and `node:fs/promises` modules, providing a sandboxed virtual file system for reading, writing, and manipulating files. [Learn more](/scripting/node/fs/node-fs).
</Update>

<Update label="January 2026" tags={["New"]}>
  ## WebSocket support

  Edge Scripts can now upgrade HTTP connections to WebSockets for real-time bidirectional communication at the edge. [Learn more](/scripting/websockets).
</Update>
