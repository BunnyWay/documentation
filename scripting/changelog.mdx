<Update label="2026-03-26" tags={["Improvement"]}>
  ## Node.js file system API is now generally available

The `node:fs` API has graduated from Preview and is now generally available, with the virtual file system memory limit removed. [Learn more](/scripting/node/fs/node-fs)

</Update>

<Update label="2026-02-11" tags={["New"]}>
  ## Node.js file system API

A Node.js-compatible file system API is now available in Edge Scripts, allowing you to read and write files directly at the edge. [Learn more](/scripting/node/fs/node-fs)

</Update>
