<Update label="February 19, 2026" tags={["New"]}>
## Terraform support for Edge Script pull zones

You can now manage the association between Edge Scripts and pull zones via Terraform, supporting both standalone and middleware script configurations.

[Learn more](/terraform/edgescript-pullzone)
</Update>

<Update label="February 11, 2026" tags={["New"]}>
## Node.js file system API

Edge Scripts now support the `node:fs` file system API with a virtual file system (64 MB limit). Use both promise-based and callback-based APIs for file and directory operations within your scripts.

[Learn more](/scripting/node/fs/node-fs)
</Update>
