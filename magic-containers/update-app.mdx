# What you'll need

Before you dive in, make sure you have the following prerequisites in place:

- A [bunny.net](https://bunny.net/) account ([Log in](https://dash.bunny.net/auth/login?pk_buttonlocation=menu) or sign up for a [free trial](https://dash.bunny.net/auth/register)).
- New Image Version. Before initiating the update, make sure you have the latest version of your application packaged as a Docker image. This image should be available in a container registry accessible to Magic Containers.

# Updating your app

To update your app to a higher version, follow the steps below: 

1. Login to [bunny.net dashboard](https://dash.bunny.net/auth/login?pk_buttonlocation=menu).

2. Navigate to the **Magic Containers** section and select the app you want to modify.

3. Click on **Container Settings** and click **Edit**.

<Frame>
![](/images/docs/78520bc0c71f7abcae3b4b337cb5d555eaf8f1e05c56f1dafb1f46191747305e-image.png)
</Frame>

5. Select the new app version from the image dropdown and click **Update Container**.
