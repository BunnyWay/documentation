<Update label="2026-03-10" tags={["New"]}>
  ## Quick Deploy

  A new streamlined deployment flow lets you go from zero to deployed in seconds. Quick Deploy presents everything in a single form, from picking your image to setting environment variables. [Learn more](/magic-containers/quick-deploy).
</Update>

<Update label="2026-03-10" tags={["New"]}>
  ## Templates

  Deploy pre-built application templates with just a few clicks. Each template includes a ready-to-run container image, and some include a sidecar database for a complete setup out of the box. [Learn more](/magic-containers/templates).
</Update>

<Update label="2026-02-19" tags={["New"]}>
  ## Database tutorials

  New step-by-step tutorials for running PostgreSQL, MariaDB, Redis, and ClickHouse on Magic Containers with persistent volumes. [Learn more](/magic-containers/guides/postgresql).
</Update>

<Update label="2026-02-09" tags={["New"]}>
  ## Graceful shutdown

  Magic Containers now supports graceful shutdown with configurable grace periods. When a container needs to stop, it receives a SIGTERM signal and has time to clean up resources and complete in-flight requests. [Learn more](/magic-containers/graceful-shutdown).
</Update>

<Update label="2026-01-30" tags={["New"]}>
  ## Public API

  The Magic Containers API is now publicly available, allowing you to deploy and manage containerized applications programmatically. [Learn more](/magic-containers/api-reference).
</Update>
