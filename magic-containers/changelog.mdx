<Update label="March 16, 2026" tags={["New"]}>
  ## WordPress deployment guide

  A step-by-step tutorial is now available for deploying WordPress with MariaDB on Magic Containers, covering both template-based and manual setup. [Learn more](/magic-containers/guides/wordpress)
</Update>

<Update label="March 10, 2026" tags={["New"]}>
  ## Templates gallery

  Deploy pre-built application templates directly from the dashboard. Choose from Django, Flask, Go, Hono, Laravel, Next.js, Rails, WordPress, and more to get started in seconds. [Learn more](/magic-containers/templates)
</Update>

<Update label="March 10, 2026" tags={["New"]}>
  ## Quick Deploy

  A new streamlined deployment flow lets you go from zero to deployed in seconds with a single form. Pick your image, configure environment variables, and deploy without stepping through multiple screens. [Learn more](/magic-containers/quick-deploy)
</Update>

<Update label="February 17, 2026" tags={["New"]}>
  ## Database deployment guides

  New tutorials are available for deploying MariaDB, PostgreSQL, ClickHouse, and Redis on Magic Containers with persistent volumes. [Learn more](/magic-containers/guides/mariadb)
</Update>

<Update label="February 9, 2026" tags={["New"]}>
  ## Graceful shutdown

  Magic Containers now supports configurable grace periods for container shutdown. Applications receive a SIGTERM signal before being stopped, giving them time to finish in-flight requests and clean up resources. [Learn more](/magic-containers/graceful-shutdown)
</Update>
