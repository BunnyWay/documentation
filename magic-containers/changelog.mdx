<Update label="2026-03-10" tags={["New"]}>
  ## Quick Deploy

  Deploy containerized applications globally in seconds with a streamlined, single-form flow. Go from selecting an image to a running deployment without stepping through multiple configuration screens. [Learn more](/magic-containers/quick-deploy)
</Update>

<Update label="2026-03-10" tags={["New"]}>
  ## Templates

  Deploy pre-built application templates with just a few clicks. Templates include ready-to-run container images for popular frameworks and stacks, with some including sidecar databases. [Learn more](/magic-containers/templates)
</Update>

<Update label="2026-02-19" tags={["New"]}>
  ## Database tutorial guides

  Step-by-step guides for running PostgreSQL, MariaDB, Redis, and ClickHouse on Magic Containers with persistent storage. [Learn more](/magic-containers/guides/postgresql)
</Update>

<Update label="2026-02-09" tags={["New"]}>
  ## Graceful shutdown

  Magic Containers now supports configurable graceful shutdown with SIGTERM signals and grace periods, giving your applications time to clean up resources and complete in-flight requests during rolling updates and scaling events. [Learn more](/magic-containers/graceful-shutdown)
</Update>
