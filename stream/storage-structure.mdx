---
title: "Storage Structure"
description: "All videos on Bunny Stream are processed and stored in a standardized location. The following page describes how and where videos are stored, allowing you to access them programmatically or within your own video player."
---

Each Bunny Stream video can be reached through several stable URL patterns, depending on whether you want the direct play URL, the original file, HLS, or per-resolution files. Use the patterns below to access videos programmatically or in your own player.

#### Direct play URL

```bash
https://video.bunnycdn.com/play/{video_library_id}/{video_id}
```

#### Original file (if enabled)

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/original
```

#### HLS playlist URL

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/playlist.m3u8
```

#### Custom thumbnail URL

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/{thumbnail_filename}
```

#### Default thumbnail URLs

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/thumbnail.jpg
```



#### Player seek thumbnail URL

<Info>
  Player seek thumbnails are visible within the video player timeline. They are incrementally generated based on video length every 2 seconds, or every 1 second for short videos.
</Info>

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/seek/{_0}.jpg
```

#### Player seek thumbnail URL examples 
```bash
  https://{pull\_zone\_url}.b-cdn.net/{video\_id}/seek/_0.jpg
  https://{pull\_zone\_url}.b-cdn.net/{video\_id}/seek/_1.jpg
  https://{pull\_zone\_url}.b-cdn.net/{video\_id}/seek/_2.jpg
  https://{pull\_zone\_url}.b-cdn.net/{video\_id}/seek/_3.jpg
```


#### Preview animation URL (WebP & Gif)

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/preview.webp
```

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/preview.gif
```

#### Preview animation URL high quality (WebM & Mp4)

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/preview_hq.webm
```

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/preview_hq.mp4
```

#### MP4 video URL

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/play_{resolution_height}p.mp4
```

#### Subtitle file URL

```bash
https://{pull_zone_url}.b-cdn.net/{video_id}/captions/{language_code}.vtt
```
