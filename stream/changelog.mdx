<Update label="2026-03-18" tags={["New"]}>
  ## Vimeo2Bunny migration tool

  Migrate your videos from Vimeo to Bunny Stream using the new Vimeo2Bunny CLI tool. Videos transfer directly via Fetch Video URL without downloading to your local machine. [Learn more](/stream/vimeo2bunny).
</Update>

<Update label="2026-03-16" tags={["New"]}>
  ## Webhook signature validation

  Verify that incoming webhook requests are authentic and untampered using HMAC-SHA256 signature validation with your library's Read-Only API key. [Learn more](/stream/webhooks).
</Update>

<Update label="2026-03-12" tags={["New"]}>
  ## Player 2.0

  The new Bunny Player 2.0 is now the default for all new video libraries. It features a refreshed interface with improved playback performance. Existing libraries can switch by disabling the legacy player toggle. [Learn more](/stream/player).
</Update>

<Update label="2026-02-11" tags={["New"]}>
  ## Mobile SDK token authentication

  Secure video playback in mobile apps using Embed View Token Authentication. Support for signed URLs, expiring tokens, and pre-signed keys is now available. [Learn more](/stream/mobile-sdk-token-authentication).
</Update>
