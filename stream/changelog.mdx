<Update label="March 2026" tags={["New"]}>
  ## Vimeo2Bunny migration tool

  A new migration tool makes it easy to move your video library from Vimeo to Bunny Stream with minimal effort.

  [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="March 2026" tags={["New"]}>
  ## Webhook signature validation

  You can now verify the authenticity of Stream webhook payloads using signature validation, ensuring your integrations only process legitimate events.

  [Learn more](/stream/webhooks)
</Update>

<Update label="March 2026" tags={["New"]}>
  ## Player 2.0

  The Bunny Player has been upgraded to version 2.0, featuring a refreshed interface, improved embedding options, and updated player settings.

  [Learn more](/stream/player)
</Update>

<Update label="March 2026" tags={["New"]}>
  ## Custom head HTML migration guide

  A new guide helps you transition to the updated custom head HTML approach for the Bunny Player.

  [Learn more](/stream/custom-head-html-migration-guide)
</Update>

<Update label="March 2026" tags={["New"]}>
  ## Smart chapters

  Automatically generate chapter markers for your videos, making it easier for viewers to navigate long-form content.

  [Learn more](/stream/smart-chapters)
</Update>

<Update label="March 2026" tags={["New"]}>
  ## Mobile SDK token authentication

  Token authentication is now available for the Stream Mobile SDK, allowing you to secure video playback in your mobile apps.

  [Learn more](/stream/mobile-sdk-token-authentication)
</Update>

<Update label="February 2026" tags={["Improvement"]}>
  ## Smart moments

  Smart Chapters now also supports Smart Moments, following the same workflow for generating key moments in your videos.

  [Learn more](/stream/smart-chapters)
</Update>

<Update label="February 2026" tags={["Improvement"]}>
  ## Remote video fetch authentication

  You can now authenticate remote video fetches using basic auth and static API keys via custom headers.

  [Learn more](/stream/url-fetch)
</Update>

<Update label="February 2026" tags={["Improvement"]}>
  ## MP4 download 1080p support

  MP4 fallback and download options now support up to 1080p resolution.

  [Learn more](/stream/mp4-downloads)
</Update>

<Update label="February 2026" tags={["Improvement"]}>
  ## Video fetch API rate limits

  Rate limit information is now documented for the Stream video HTTP fetch API, helping you plan your integration accordingly.

  [Learn more](/stream/http-api)
</Update>
