<Update label="March 20, 2026" tags={["Improvement"]}>
  ## Transcription language detection

  Transcription now automatically detects the spoken language from the first 30 seconds of audio. You can also set the language manually when your video doesn't contain speech in the opening moments. [Learn more](/stream/transcribing)
</Update>

<Update label="March 18, 2026" tags={["New"]}>
  ## Vimeo2Bunny CLI

  A new CLI tool makes it easy to migrate your video library from Vimeo to Bunny Stream. Videos are transferred directly using Fetch Video URL with no local download required. [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="March 16, 2026" tags={["Improvement"]}>
  ## Webhook signature validation

  You can now verify that incoming webhook requests are genuinely from Bunny Stream by validating the request signature. [Learn more](/stream/webhooks)
</Update>

<Update label="March 12, 2026" tags={["New"]}>
  ## Player 2.0

  The new Bunny Stream Player is now available with a modern interface and improved performance. The updated player offers a cleaner design, faster load times, and better playback experience. [Learn more](/stream/player)
</Update>

<Update label="March 5, 2026" tags={["New"]}>
  ## Custom head HTML migration guide

  A new migration guide is available to help you transition custom head HTML markup from the legacy player to the new Web Components-based player architecture. [Learn more](/stream/custom-head-html-migration-guide)
</Update>

<Update label="February 23, 2026" tags={["Improvement"]}>
  ## Remote video fetch authentication

  You can now authenticate remote video fetches using basic auth and custom headers for static API keys, making it easier to import videos from protected sources. [Learn more](/stream/http-api)
</Update>

<Update label="February 19, 2026" tags={["Improvement"]}>
  ## Smart moments

  Smart Chapters now includes Smart Moments, letting you automatically generate key moments from your video's caption track using AI. [Learn more](/stream/smart-chapters)
</Update>

<Update label="February 11, 2026" tags={["New"]}>
  ## Mobile SDK token authentication

  Token authentication is now available for the Bunny Stream Mobile SDKs, providing secure playback with backend-generated embed view tokens. [Learn more](/stream/mobile-sdk-token-authentication)
</Update>

<Update label="January 23, 2026" tags={["New"]}>
  ## Smart chapters

  Generate chapters automatically using your video's caption track and AI. Enable Smart Chapters in your library settings to automatically create chapters for every new upload. [Learn more](/stream/smart-chapters)
</Update>
