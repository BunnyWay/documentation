<Update label="2026-03-18" tags={["New"]}>
  ## Vimeo2Bunny migration tool

  Migrate your video library from Vimeo to bunny.net Stream using the Vimeo2Bunny CLI. Videos transfer directly from Vimeo to bunny.net without downloading to your machine. [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="2026-03-16" tags={["New"]}>
  ## Webhook signature validation

  Verify the authenticity of Stream webhook callbacks using HMAC-SHA256 signature validation. This ensures webhook payloads are genuinely from bunny.net and haven't been tampered with. [Learn more](/stream/webhooks)
</Update>

<Update label="2026-03-12" tags={["New"]}>
  ## Player 2.0

  The new Bunny Stream video player is here with a modern interface and improved performance. Player 2.0 is enabled by default for all new video libraries and can be activated for existing libraries in your player settings. [Learn more](/stream/player)
</Update>

<Update label="2026-02-11" tags={["New"]}>
  ## Mobile SDK token authentication

  Secure video playback in mobile apps with embed view token authentication. Use a customer-managed backend or Edge Script to control which clients can play videos through the Stream Mobile SDKs. [Learn more](/stream/mobile-sdk-token-authentication)
</Update>
