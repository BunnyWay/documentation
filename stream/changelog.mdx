<Update label="2026-03-18" tags={["New"]}>
  ## Vimeo2Bunny migration tool

  A new CLI tool makes it easy to migrate your videos from Vimeo to Bunny Stream. Videos transfer directly from Vimeo to bunny.net via URL fetch, so nothing is downloaded to your machine. [Learn more](/stream/vimeo2bunny).
</Update>

<Update label="2026-03-16" tags={["New"]}>
  ## Webhook signature validation

  You can now verify the authenticity of webhook callbacks using signature validation. This ensures that incoming webhook requests are genuinely from bunny.net and haven't been tampered with. [Learn more](/stream/webhooks).
</Update>

<Update label="2026-03-12" tags={["New"]}>
  ## Player 2.0

  The new Bunny Stream video player is here with a modern interface and improved performance. Player 2.0 is enabled by default for all new video libraries and includes an updated embedding experience. [Learn more](/stream/player).
</Update>

<Update label="2026-02-11" tags={["New"]}>
  ## Mobile SDK token authentication

  You can now use token authentication with the Bunny Stream Mobile SDKs to securely control video playback access in your mobile applications. [Learn more](/stream/mobile-sdk-token-authentication).
</Update>

<Update label="2026-01-23" tags={["New"]}>
  ## Smart Chapters

  Automatically generate chapters for your videos using AI. Smart Chapters analyzes your video's caption track to create meaningful chapter markers, making it easier for viewers to navigate your content. [Learn more](/stream/smart-chapters).
</Update>

<Update label="2026-03-05" tags={["Improvement"]}>
  ## Player migration guide

  A new migration guide is available to help you transition custom `<head>` markup from the legacy player to the new Bunny Stream Player 2.0. [Learn more](/stream/custom-head-html-migration-guide).
</Update>

<Update label="2026-01-28" tags={["Improvement"]}>
  ## MP4 fallback now supports 1080p

  MP4 fallback downloads now support 1080p resolution, giving you higher quality options for environments where HLS playback isn't available. [Learn more](/stream/mp4-downloads).
</Update>
