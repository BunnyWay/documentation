<Update label="2026-04-22" tags={["Improvement"]}>
  ## Transcribing language model upgrade

The transcribing language model has been upgraded to v1.2.0, offering faster and higher quality transcriptions. [Learn more](/stream/transcribing)

</Update>

<Update label="2026-04-22" tags={["Improvement"]}>
  ## Stream API enhancements

The Stream API now features an upgraded Smart feature language model for better generation quality and improved JIT watermarking reliability. [Learn more](/changelog)

</Update>

<Update label="2026-04-22" tags={["Improvement"]}>
  ## Enhanced bunny player performance

The bunny player now offers smoother and more reliable playback with upgrades to media-chrome, improved Firefox H.265 seeking, and refined iOS fullscreen transitions. [Learn more](/changelog)

</Update>

<Update label="2026-03-27" tags={["New"]}>
  ## Free encoding

  Increased capacity and performance improvements to Stream Free encoding for faster transcoding and shorter queue time. [Learn more](/stream/encoding)
</Update>

<Update label="2026-03-18" tags={["New"]}>
  ## Vimeo2Bunny CLI

  A new command-line tool to migrate videos from Vimeo to Bunny Stream. Videos transfer directly from Vimeo via Fetch Video URL — nothing is downloaded to your machine. [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="2026-03-16" tags={["New"]}>
  ## Webhook signature validation

  You can now validate webhook signatures to verify that incoming webhook notifications are genuinely from Bunny Stream and have not been tampered with. [Learn more](/stream/webhooks)
</Update>

<Update label="2026-03-12" tags={["New"]}>
  ## Player 2.0

  The new Bunny Stream video player is here with a modern interface and improved performance. It's enabled by default for all new video libraries. [Learn more](/stream/player)
</Update>

<Update label="2026-02-11" tags={["New"]}>
  ## Mobile SDK token authentication

  Token authentication is now available for the Stream Mobile SDKs, adding an extra layer of security for mobile video playback. [Learn more](/stream/mobile-sdk-token-authentication)
</Update>

<Update label="2026-01-23" tags={["New"]}>
  ## Smart Chapters

  Automatically generate chapters for your videos using AI. Smart Chapters analyzes your video's caption track to create chapter markers, improving navigation for viewers. [Learn more](/stream/smart-chapters)
</Update>
