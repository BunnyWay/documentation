<Update label="2026-03-20" tags={["Stream", "Improvement"]}>
  ## Transcription language detection

  Bunny Stream now automatically detects the spoken language in your videos based on the first 30 seconds of audio, making transcription setup even easier.

  [Learn more](/stream/transcribing)
</Update>

<Update label="2026-03-18" tags={["CDN", "New"]}>
  ## Edge rule pattern matching

  You can now use Lua-based pattern matching in CDN Edge Rule conditions to match structured request values such as URLs, headers, cookies, and query strings for advanced traffic routing and security logic.

  [Learn more](/cdn/edge-rules/pattern-matching)
</Update>

<Update label="2026-03-18" tags={["Stream", "New"]}>
  ## Vimeo2Bunny migration tool

  A new CLI tool makes it easy to migrate your video library from Vimeo to Bunny Stream. Videos are transferred directly without downloading to your local machine.

  [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="2026-03-17" tags={["Database", "New"]}>
  ## Database Shell (bsql)

  An interactive SQL terminal client is now available for querying and managing your Bunny Database with dot-commands, multiple output formats, and persistent history.

  [Learn more](/database/connect/database-shell)
</Update>

<Update label="2026-03-16" tags={["Stream", "New"]}>
  ## Webhook signature validation

  You can now verify the authenticity of incoming Bunny Stream webhook requests using HMAC-SHA256 signature validation, ensuring your webhook payloads haven't been tampered with.

  [Learn more](/stream/webhooks)
</Update>

<Update label="2026-03-12" tags={["Stream", "New"]}>
  ## Player 2.0

  The new Bunny Stream player is here with a modern interface, improved performance, and enhanced customization options.

  [Learn more](/stream/player)
</Update>
