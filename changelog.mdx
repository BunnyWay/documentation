<Update label="2026-03-20" tags={["Improvement", "Stream"]}>
  ## Transcribing language detection

  Transcribing now includes details on how automatic language detection works, helping you understand how audio is analyzed during the first 30 seconds of your video. [Learn more](/stream/transcribing)
</Update>

<Update label="2026-03-18" tags={["New", "CDN"]}>
  ## Edge Rule pattern matching

  You can now use Lua-based pattern matching in Edge Rule conditions to match structured request values such as URLs, headers, cookies, and query strings. [Learn more](/cdn/edge-rules/pattern-matching)
</Update>

<Update label="2026-03-18" tags={["New", "Stream"]}>
  ## Vimeo2Bunny migration tool

  A new CLI tool that lets you migrate videos directly from Vimeo to Bunny Stream without downloading files locally. [Learn more](/stream/vimeo2bunny)
</Update>

<Update label="2026-03-17" tags={["New", "Database"]}>
  ## Database Shell

  Connect to your Bunny Database using `bsql`, a standalone interactive SQL shell with dot-commands, multiple output formats, and persistent history. [Learn more](/database/connect/database-shell)
</Update>

<Update label="2026-03-16" tags={["New", "Stream"]}>
  ## Webhook signature validation

  You can now verify the authenticity of Stream webhook payloads using HMAC-SHA256 signatures sent in the `X-BunnyStream-Signature` header. [Learn more](/stream/webhooks)
</Update>

<Update label="2026-03-12" tags={["New", "Stream"]}>
  ## Player 2.0

  The new Bunny Stream video player is now the default, featuring a modern interface and improved performance across all video libraries. [Learn more](/stream/player)
</Update>

<Update label="2026-03-10" tags={["New", "Magic Containers"]}>
  ## Templates

  Deploy pre-built application templates with a single click in Magic Containers, including options with sidecar databases for common stacks. [Learn more](/magic-containers/templates)
</Update>

<Update label="2026-03-10" tags={["New", "Magic Containers"]}>
  ## Quick Deploy

  A streamlined single-form deployment flow that lets you deploy containerized applications globally in seconds. [Learn more](/magic-containers/quick-deploy)
</Update>

<Update label="2026-03-09" tags={["New", "CDN"]}>
  ## JA4 fingerprinting

  Identify clients using TLS fingerprinting with JA4 support. Access the JA4 fingerprint through the `CDN-JA4` request header for bot detection, DDoS mitigation, and advanced security decisions. [Learn more](/cdn/security/ja4-fingerprinting)
</Update>

<Update label="2026-03-09" tags={["Improvement", "CDN"]}>
  ## Logging updates

</Update>
