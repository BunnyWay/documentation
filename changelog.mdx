<Update label="2026-03-18" tags={["New", "Stream"]}>
  ## Vimeo2Bunny migration tool

  Migrate your videos from Vimeo to Bunny Stream using the new Vimeo2Bunny CLI tool. Videos transfer directly via Fetch Video URL without downloading to your local machine. [Learn more](/stream/vimeo2bunny).
</Update>

<Update label="2026-03-18" tags={["New", "CDN"]}>
  ## Edge Rules pattern matching

  Use Lua-based pattern matching in Edge Rule conditions to match URLs, headers, cookies, and query strings with fine-grained control. [Learn more](/cdn/edge-rules/pattern-matching).
</Update>

<Update label="2026-03-17" tags={["New", "Database"]}>
  ## Database Shell

  Manage your Bunny Database from the terminal with `bsql`, an interactive SQL shell featuring dot-commands, multiple output formats, and persistent history. [Learn more](/database/connect/database-shell).
</Update>

<Update label="2026-03-16" tags={["New", "Stream"]}>
  ## Webhook signature validation

  Verify that incoming Stream webhook requests are authentic and untampered using HMAC-SHA256 signature validation. [Learn more](/stream/webhooks).
</Update>
