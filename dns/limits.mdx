---
title: DNS Limits and Defaults
sidebarTitle: Limits
description: Overview of the default limits for Bunny DNS.
---

| Limit                 | Value | Description                                      |
| --------------------- | ----- | ------------------------------------------------ |
| Records per zone      | 5,000 | Maximum number of DNS records in a single zone.  |
| DNS zones per account | 500   | Maximum number of DNS zones across your account. |

These are default limits, designed to accommodate typical use cases under normal operating conditions.

<Info>
  These limits can be raised upon request. If you need higher limits, please [contact our sales team](https://bunny.net/contact-sales/) with your use case and expected scale.
</Info>
