<Update label="2026-03-17" tags={["New"]}>
  ## Database Shell

  Manage your database from the terminal with `bsql`, an interactive SQL shell featuring dot-commands, multiple output formats, and persistent history. [Learn more](/database/connect/database-shell).
</Update>

<Update label="2026-01-20" tags={["New"]}>
  ## SQL API

  Query your database directly over HTTP using the SQL API. [Learn more](/database/connect/sql-api).
</Update>

<Update label="2026-01-19" tags={["New"]}>
  ## SDKs for .NET, Go, and Rust

  Connect to Bunny Database from your preferred language with official SDKs for .NET, Go, and Rust. [Learn more](/database/connect/dotnet).
</Update>
