<Update label="2026-03-17" tags={["New"]}>
  ## Database Shell

The Bunny Database Shell (`bsql`) is a standalone, interactive SQL shell for querying and managing your database from the terminal. It supports dot-commands, multiple output formats, and persistent history. [Learn more](/database/connect/database-shell)

</Update>

<Update label="2026-01-19" tags={["New"]}>
  ## Database SDKs

Connect to Bunny Database using official SDKs for TypeScript, Go, Rust, and .NET. [Learn more](/database/connect/typescript)

</Update>

<Update label="2026-01-20" tags={["New"]}>
  ## SQL API

Execute SQL queries against your Bunny Database over HTTP using the SQL API. [Learn more](/database/connect/sql-api)

</Update>
