<Update label="March 17, 2026" tags={["New"]}>
## Database Shell

The Bunny Database Shell (`bsql`) is an interactive SQL shell for querying and managing your databases from the terminal. It supports dot-commands, multiple output formats, sensitive column masking, saved views, and persistent history.

[Learn more](/database/connect/database-shell)
</Update>

<Update label="January 20, 2026" tags={["New"]}>
## SQL API

You can now execute SQL queries over HTTP using the SQL API, without requiring an SDK. It uses the libSQL remote protocol (Hrana over HTTP) for direct database access.

[Learn more](/database/connect/sql-api)
</Update>
