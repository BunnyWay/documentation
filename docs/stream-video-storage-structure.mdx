# Video storage structure

### Direct play URL

<CodeGroup>
  ```bash bash https://video.bunnycdn.com/play/{video_library_id}/{video_id}
  ```
</CodeGroup>

### Original file (if enabled)

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/original ```
</CodeGroup>

### HLS playlist URL

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/playlist.m3u8 ```
</CodeGroup>

### Thumbnail URL

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/
  {thumbnail_file_name}
  ```
</CodeGroup>

### Preview animation URL (WebP)

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/preview.webp ```
</CodeGroup>

### MP4 video URL

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/play_
  {resolution_height}p.mp4 ```
</CodeGroup>

### Subtitle file URL

<CodeGroup>
  ```bash bash https://{pull_zone_url}.b-cdn.net/{video_id}/captions/
  {language_code}.vtt ```
</CodeGroup>
