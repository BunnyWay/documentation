## Brightness

**_Parameter_**: brightness **_Units_**: Number **_Range_**: -100 to 100 **_Default_**: 0

You can easily lighten or darken the image using the brightness parameter. Mind that this operation affects each pixel in the same manner so you will quickly lose the blacks when making the image brighter and whites, making it darker.

## Gamma

**_Parameter_**: gamma **_Units_**: Number **_Range_**: -100 to 100 **_Default_**: 0

Brighten or darken an image based on each pixel's current luminosity. Contrary to the brightness adjustment, with gamma, bright pixels will be affected more than dark pixels, which more closely mimics how our eyes perceive levels of brightness and can therefore expose additional detail while better-preserving blacks/whites. You can use it to fix the global tonality issues of your images.
