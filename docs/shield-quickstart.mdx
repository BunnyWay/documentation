Protecting your web applications from malicious traffic is crucial in today's digital landscape. **Bunny Shield** offers a solution to safeguard your website by filtering out unwanted traffic and providing detailed logs for analysis. This quickstart guide will walk you through the initial setup of a Shield Zone and how to review and adjust your Web Application Firewall (WAF) settings.

# What you’ll need

Before you dive in, make sure you have the following prerequisites in place:

* A [bunny.net](https://bunny.net/) account ( [Log in](https://dash.bunny.net/auth/login?pk_buttonlocation=menu) or sign up for a [free trial](https://dash.bunny.net/auth/register)).
* An existing Pull Zone you want to protect.
* Your AccessKey or JWT for API authentication.

# Creating a Shield Zone

To get started, you'll need to create a **Shield Zone**. This is where all your Bunny Shield settings and configurations reside. The Shield Zone is then associated with the Pull Zone you wish to protect.

1. Click on the **Get Started** button.

![](/images/docs/025b6da75c4b14d2da72387fc826a5773bb56130fd757240ecf945afad12ee75-image.png)

2. Select a WAF Profile and click **Next Step**, using a WAF Profile reduces the potential for false positives for your web application.

![](/images/docs/cfaf5d261dd95d5b9f2abf393e952cfbf4d86fa290038af9f39c83e754d85f5d-image.png)

3. Ensure you have choosen a plan that fits your needs, and then click **Confirm**.

![](/images/docs/c48c7fc114cbe8507c8b133bb854d8768ca797463aa3b5400d20027b50c28d98-image.png)
