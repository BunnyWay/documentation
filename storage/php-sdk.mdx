---
title: PHP SDK
description: Official PHP SDK for Bunny Storage
---

The official PHP SDK provides an easy way to interact with Bunny Storage in your PHP applications.

## Installation

```bash
composer require bunnycdn/storage
```

## Quickstart

For detailed usage examples and documentation, see the [PHP Storage Library on GitHub](https://github.com/BunnyWay/BunnyCDN.PHP.Storage).
