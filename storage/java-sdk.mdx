---
title: Java SDK
description: Official Java SDK for Bunny Storage
---

The official Java SDK provides a type-safe way to interact with Bunny Storage in your Java applications.

## Installation

Add to your `pom.xml`:

```xml
<dependency>
  <groupId>net.bunny</groupId>
  <artifactId>storage</artifactId>
  <version>1.0.0</version>
</dependency>
```

## Quickstart

For detailed usage examples and documentation, see the [Java Storage Library on GitHub](https://github.com/BunnyWay/BunnyCDN.Java.Storage).
