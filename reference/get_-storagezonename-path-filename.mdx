---
title: "Download File"
description: "get https://storage.bunnycdn.com/{storageZoneName}/{path}/{fileName}"
---

Returns the stored file at the given path. If the file does not exist, a 404 response will be returned.

⚠️ The storage endpoint hostname will depend on the primary region of the storage zone. For determining the storage zone hostname, please check the [storage endpoints documentation](/reference/storage-api#storage-endpoints).
