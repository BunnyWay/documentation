## Stream

![](/images/reference/661db18-bunnynet-logo-dark-small.png "bunnynet-logo-dark-small.png")

Learn how to use the [bunny.net](https://bunny.net "bunny.net - The content delivery platform that truly hops") Stream API. It allows you to upload, manage and process videos with a few lines of code.

# Authentication

### Header name: **AccessKey**

The API requests are authenticated using the **AccessKey** header. You can find it in your Video Library details page in the bunny.net dashboard.

<Warning>
  The Stream API uses its own authentication keys. Authenticating via the bunny.net account API key will not work.
</Warning>

# Third Party API Clients

We currently do not maintain an official API library, but you can use one of the third-party options provided here:

* [bunny-sdk](https://bunny-launcher.net/#bunny-sdks) by jlarmstrongiv

<Warning>
  These third-party clients are not maintained or developed by bunny.net. We cannot offer support for them or confirm the security of these libraries.
</Warning>
