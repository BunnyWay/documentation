<Update label="2026-04-28" tags={["Improvement"]}>
  ## Image upscaling option

Introduced the 'upscaling' parameter to allow images to be enlarged beyond their original dimensions using resampling. [Learn more](/optimizer/dynamic-images/resizing)

</Update>

<Update label="2026-04-28" tags={["Improvement"]}>
  ## AVIF input format support

Bunny Optimizer now supports AVIF as an input format, allowing for efficient image compression using AV1. [Learn more](/optimizer/dynamic-images/formats)

</Update>
