The official [bunny.net WordPress plugin](https://wordpress.org/plugins/bunnycdn/) integrates bunny.net's delivery optimization services into your WordPress site. Install the plugin, connect your account, and start delivering content faster with just a few clicks.

## What's included

- **Bunny CDN** — Rewrites static content links with CDN URLs to improve loading times
- **Bunny Optimizer** — Compresses files and images to reduce file size
- **Bunny Offloader** — Transfers media files to Bunny Storage with multi-region replication
- **Bunny Stream** — Upload and embed videos with a dedicated block and shortcode
- **Bunny Fonts** — GDPR-compliant fonts hosted within the EU, with no tracking
