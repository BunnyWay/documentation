The official bunny.net WordPress plugin makes it easy to integrate your WordPress site with bunny.net services. Install the plugin, connect your account, and start delivering content faster with just a few clicks.

## What's included

- **CDN acceleration** with automatic URL rewriting for static assets
- **Content offloading** to Bunny Storage with multi-region replication
- **One-click setup** via the built-in Integration Wizard
- **Bunny DNS integration** for full-stack acceleration
