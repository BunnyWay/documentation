**Bunny Shield** protects your site from attacks, malicious traffic, and service disruption using a built-in Web Application Firewall (WAF) and DDoS mitigation.

<Note>
Shield requires Bunny DNS with CDN Acceleration enabled. If you haven't set this up yet, follow [Enable Bunny DNS and CDN Acceleration](/integrations/wordpress/dns) first.
</Note>

## Enable Bunny Shield

In your WordPress Admin Panel, navigate to **bunny.net → Shield**. If DNS and CDN Acceleration are not yet configured, the Shield tab prompts you to enable them first. The **Enable Bunny DNS** button takes you through the same setup covered in [Enable Bunny DNS and CDN Acceleration](/integrations/wordpress/dns).

<Frame>
  ![Bunny Shield tab in the WordPress plugin](/images/integrations/wordpress/shield-enable-bunny-dns.png)
</Frame>

Once acceleration is active, the Shield tab exposes **WAF** and **DDoS** settings, which you can configure to protect your site.
