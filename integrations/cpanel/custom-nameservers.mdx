---
title: "Custom nameservers"
description: "Learn how to configure custom nameservers for the bunny.net cPanel plugin."
---

By default the bunny.net cPanel plugin uses bunny.net nameservers and displays them in the plugin. If you want to use custom nameservers, you need to update your DNS configuration and add or edit the `/etc/bunnynet.conf` file.

1. Update the example.com DNS zone, pointing the addresses to bunny.net:
   ```
   ns1.example.com.  IN  A     91.200.176.1
   ns1.example.com.  IN  AAAA  2400:52e0:fff0::1
   ns2.example.com.  IN  A     109.104.147.1
   ns2.example.com.  IN  AAAA  2400:52e0:fff2::1
   ```
2. On your cPanel server, make sure those lines exist in your `/etc/bunnynet.conf` file:
   ```
   NS1=ns1.example.com
   NS2=ns2.example.com
   SOA_EMAIL=hostmaster@example.com
   ```
3. Run `systemctl restart bunnynet-cpanel-daemon`;
