## APT

Configure the APT repository:

```bash
cat <<EOF > /etc/apt/auth.conf.d/bunnynet-cpanel.conf
machine https://cpanel-repo.bunny.net
login username-goes-here
password password-goes-here
EOF
```

## RPM

```bash
cat <<EOF > /etc/yum.repos.d/bunnynet-cpanel.repo
[bunnynet-cpanel]
name=bunny.net cPanel Plugin
baseurl=https://cpanel-repo.bunny.net/rpm
gpgcheck=1
gpgkey=https://cpanel-repo.bunny.net/gpg.key
username=username-goes-here
password=password-goes-here
EOF
```

Then install the packages:

```bash
dnf install bunnynet-cpanel bunnynet-cpanel-daemon
```
