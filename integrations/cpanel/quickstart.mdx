This guide walks you through installing and setting up the bunny.net cPanel plugin on your server.

## Prerequisites

- cPanel/WHM server with root access
- A bunny.net account with an API key

## Install the plugin

    login username-goes-here
    password password-goes-here
    EOF

## Set up the plugin

After installation, open WHM and navigate to the **bunny.net cPanel plugin** page.

<Steps>
  <Step title="Set up the DNS cluster">
    The plugin requires a DNS cluster to be configured. Click **Setup DNS cluster** to open the DNS Cluster configuration page.

    <Warning>
      The DNS Role must be set to **write-only**.
    </Warning>

    <Frame>
      ![bunny.net cPanel plugin requirements](/images/integrations/cpanel/setup_step1.png)
    </Frame>
  </Step>
  <Step title="Add a server to the DNS cluster">
    On the DNS Cluster page, select **cPanel** as the Backend Type and click **Configure**.

    <Frame>
      ![DNS Cluster setup](/images/integrations/cpanel/setup_step2.png)
    </Frame>
  </Step>
  <Step title="Configure the DNS cluster">
    Enter your **bunny.net API key** and set the **DNS Role** to **Standalone**.

    You can also enable **Sync all DNS zones** to automatically synchronize all existing DNS zones to the cluster.

    You can find your API key in the [bunny.net dashboard](https://dash.bunny.net).

    <Frame>
      ![DNS Cluster configuration](/images/integrations/cpanel/dns-cluster-config.png)
    </Frame>
  </Step>
  <Step title="Enter your bunny.net API key">
    Back on the plugin page, enter your **bunny.net API key** and click **Setup**.

    <Frame>
      ![bunny.net API key entry](/images/integrations/cpanel/setup_step4.png)
    </Frame>
  </Step>
  <Step title="Setup complete">
    The setup is complete. The plugin will display your API keys and the available pricing zones.

    <Note>
      The bunny.net API key also needs to be updated on the DNS cluster. If you see a warning about this, update it in **Clusters > DNS Cluster**.
    </Note>

    <Frame>
      ![Setup complete](/images/integrations/cpanel/setup_step5.png)
    </Frame>
  </Step>
</Steps>

After setup, you can configure bunny.net features per hosting package under **Packages > Edit a Package**.
